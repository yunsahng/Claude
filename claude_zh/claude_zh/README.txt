# Claude 界面汉化工具

将 Claude 桌面版界面全面汉化（菜单、设置、对话框、连接器、模型选择等）。

## 适用版本

Windows 版 Claude 桌面版（从 claude.com/download 或 Microsoft Store 安装）

## 使用方法

### 准备工作

1. 电脑上已安装 **Claude 桌面版**
2. 电脑上已安装 **Node.js**（[nodejs.org](https://nodejs.org/)，下载 LTS 版本安装即可）
3. 双击前请**完全退出 Claude**（右下角系统托盘图标上右键 → 退出）

### 一键汉化

双击 **`run.bat`**，按提示操作：

```
① 自动检测 Claude 安装位置
② 自动生成汉化文件
③ 弹窗请求管理员权限 → 点击「是」
④ 完成！重启 Claude 即可看到中文界面
```

整个过程约 **10 秒**。

### 注意事项

- 首次运行会弹出 **UAC 用户账户控制** 窗口，请点击「是」——这是因为需要向 WindowsApps 受保护目录写入文件
- **以后 Claude 更新后**，重新运行一次 `run.bat` 即可
- 本工具**不影响 Claude 的功能**，仅替换界面的文字显示
- 卸载汉化：重新安装 Claude 即可恢复英文

## 文件说明

| 文件 | 用途 |
|------|------|
| `run.bat` | 一键运行脚本（双击这个） |
| `translate.js` | 核心汉化引擎（含完整翻译表） |

## 常见问题

**Q: 双击 run.bat 后黑框一闪而过？**
A: 在 `run.bat` 上右键 → 「以管理员身份运行」。或者先打开 CMD，拖入 `run.bat` 回车。

**Q: 提示"未找到 Node.js"？**
A: 去 https://nodejs.org 下载安装 Node.js，安装后重启电脑再试。

**Q: 汉化后部分区域还是英文？**
A: 完全退出 Claude（包括系统托盘图标），再重新打开。如果仍有英文，再次运行 `run.bat`。

**Q: Claude 更新后汉化失效了？**
A: Claude 更新会覆盖语言文件，重新运行一次 `run.bat` 即可。

**Q: 如何恢复英文？**
A: 重新安装 Claude 即可，或者在设置中切换语言（如 Claude 支持多语言切换）。
